由於大學履歷蠻空的，所以主要在摸索呈現方式。

* Activity方面是有動畫的但是一進入網頁就有動畫了所以滑到下面的時候動畫已經結束了。
* Interest是覺得不太重要所以不想直接放在頁面所以用另外的視窗呈現。
